Authorship

Create books and authors with this simple angular app

View on GitHub Download .zip Download .tar.gz

Overview

This Simple front end app uses the angular framework, bootstrap css and custom css for styling, npm for package management and grunt for building.

Usage

Use this app to create books and authors. You cannot delete an author that has books associated with it. There are 3 default books and 4 default authors.
